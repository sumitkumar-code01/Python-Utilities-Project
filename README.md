# Python Utilities Challenge 

Welcome to my Python learning journey! This repository contains a collection of utility scripts and mini-projects developed as part of a daily coding challenge. Each script solves a specific real-world problem or automates a simple task.

# Projects Included

Here is the list of utilities developed from PP 1 to PP 9. 
Each Project focuses on a unique use case:

* **PP 1: Self-Intro Script Generator** - Automates personalized introductions.
* **PP 2: Stylish Bio Generator** - Creates cool bios for Instagram/Twitter.
* **PP 3: Simple Bill Splitter** - Easily calculates shares among friends.
* **PP 4: Minutes Alive Calculator** - Calculates your age in total minutes.
* **PP 5: Emoji Enhancer** - Adds flair to your messages with emojis.
* **PP 6: Daily Learning Journal Logger** - A simple tool to log daily progress.
* **PP 7: Terminal-Based Task List Manager** - A CLI tool to manage your To-Do list.
* **PP 8: Password Strength Checker & Suggestion Tool** - Analyzes and suggests better passwords.
* **PP 9: Countdown Timer** - A simple functional timer script.
* **PP 10:Secret Message Encryptor & Decryptor** - Encrypted and Decrypted message.
